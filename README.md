# 🏫 401教室使用情况 - 使用说明

## 📌 快速开始

### 方式一：双击启动（推荐 Windows 用户）
1. 双击 `启动系统.bat`
2. 等待浏览器自动打开
3. 开始使用！

### 方式二：命令行启动
```bash
# Windows
python -m streamlit run classroom_booking.py

# Mac/Linux
python3 -m streamlit run classroom_booking.py
```

## 🔐 登录说明

**默认管理员密码：`123456`**

首次使用请及时修改密码：
1. 用记事本打开 `classroom_booking.py`
2. 找到第 `ADMIN_PASSWORD = "123456"` 这一行
3. 把 `123456` 改成你的新密码

## 📋 功能说明

### 1️⃣ 查看预约情况（所有用户）
- 打开页面即可看到当前周的教室使用情况
- 使用左侧 ◀ ▶ 按钮切换周
- 点击「📆 回本周」快速回到当前周
- 绿色 = 空闲，黄色 = 已预约

### 2️⃣ 添加预约（管理员）
1. 在左侧输入密码登录
2. 填写预约信息：
   - 📅 日期：选择预约日期
   - ⏰ 时间段：选择时间段（08:00-22:00，每30分钟一段）
   - 🏫 教室：选择教室
   - 👤 使用人：输入使用人姓名
3. 点击「确认添加」

### 3️⃣ 删除预约（管理员）
1. 登录后，展开左侧「🗑️ 删除预约」
2. 选择要删除的日期、时间段、教室
3. 点击「确认删除」

## ☁️ 云端部署（可选）

如果你希望同事可以直接通过网页访问系统，而不是只能在你电脑上查看：

### 部署步骤：

1. **注册 GitHub 账号**（如果没有）
   - 访问 https://github.com
   - 点击 Sign up 注册

2. **创建新仓库**
   - 点击右上角 + → New repository
   - 仓库名称：`classroom-booking`
   - 选择 Private（私有）
   - 点击 Create repository

3. **上传代码**
   - 在仓库页面点击 "uploading an existing file"
   - 把 `classroom_booking.py` 和 `reservations.csv` 两个文件拖入上传
   - 点击 Commit changes

4. **部署到 Streamlit Cloud**
   - 访问 https://streamlit.io/cloud
   - 点击 "Connect to GitHub"
   - 授权访问
   - 点击 "New app"
   - 选择你的仓库和 main branch
   - Main file path: `classroom_booking.py`
   - 点击 "Deploy!"

5. **等待部署完成**（约2-3分钟）
   - 部署成功后，你会获得一个链接，如：
   - `https://classroom-booking.streamlit.app`
   - 把这个链接分享给同事即可！

6. **重要：设置密码保护**
   - 部署后点击 Advanced settings
   - 勾选 Enable password protection
   - 设置你想要的访问密码
   - 这样只有知道密码的人才能登录管理

## 🔧 常见问题

### Q: 启动后浏览器没反应？
A: 检查是否有防火墙拦截，允许 Streamlit 通过即可

### Q: 想修改默认密码？
A: 用记事本打开 `classroom_booking.py`，修改第 `ADMIN_PASSWORD = "123456"` 行

### Q: 想增加/减少时间段？
A: 用记事本打开 `classroom_booking.py`，修改 `TIME_SLOTS` 列表

### Q: 数据丢失了怎么办？
A: 如果有备份的 `reservations.csv`，重新放到同一文件夹即可
